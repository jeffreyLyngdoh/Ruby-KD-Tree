This mini project implmentated a entented KD tree in ruby based on the definition provided in the pdfs.
Brief TLDR of a KD tree is a data structure that stores multiple dimension. A standard binary tree is stuctured based on a speific value it is 1 dimentional, an extended kd tree may repersent 2 points on a graph, 3 points on a 3d graph, etc. K dimentions total.
